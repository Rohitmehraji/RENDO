# RENDO — The Infinite Battle Reality

RENDO is an AI-first persistent battle + life simulation universe built as a global phenomenon product: **game + movie + social life + living AI world**.

## What this repo now contains

- Global-phenomenon GDD and architecture blueprint
- Microservice design for cinematic, memory, and emotion systems
- OpenAPI contracts for highlights and voice-to-world commands
- Production-oriented FastAPI service scaffold
- Kubernetes deployment baseline including Neo4j-style memory graph runtime

## Core viral pillars

1. **Hollywood Moment Engine**: auto-detected cinematic moments and share-ready reels.
2. **AI Memory World**: NPC/city/faction memory creates persistent world history.
3. **Emotional Companions**: AI teammates with relationship dynamics.
4. **Creator Mode**: instant content generation and export.

## Build Program (High Level)

1. **Phase 1:** combat + clone + cinematic highlights + voice commands.
2. **Phase 2:** companion AI + live global events + social economy.
3. **Phase 3:** self-evolving world simulation at global scale.

## Ethical & Safety Guardrails

- No copyrighted likeness cloning.
- Consent-gated biometric pipelines.
- Anti-toxicity and player wellbeing protections.
- Human policy gates for autonomous balance/content updates.
